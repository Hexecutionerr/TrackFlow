# 🚀 TrackFlow – Real-Time Location Tracking System

TrackFlow is a real-time location tracking web application built using 
Node.js, Express, Socket.io, and Leaflet.js.

It enables live tracking of multiple users on an interactive map with 
real-time updates using WebSockets.

---

## 🌟 Features

- 📍 Real-time location tracking
- 🔄 Live marker updates
- 👥 Multiple user tracking
- ❌ Automatic marker removal on disconnect
- 🗺️ Interactive map using Leaflet.js
- ⚡ WebSocket-based instant communication

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- Socket.io
- Leaflet.js
- EJS
- HTML, CSS, JavaScript

## 🚀 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Hexecutionerr/TrackFlow.git
cd TrackFlow

Install dependencies:

npm install

Run the server:

node app.js

Open browser:

http://localhost:3000
📌 How It Works

The browser fetches the user's location using Geolocation API.

Location data is sent to the server via Socket.io.

The server broadcasts location updates to all connected users.

Map markers update in real-time.
